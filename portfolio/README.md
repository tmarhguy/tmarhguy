## About

A modern, performant portfolio site showcasing hardware projects, software work, certifications, and professional background. Built for clarity, speed, and SEO — from the 3,488-transistor ALU to ASIC tapeouts and full-stack applications.

- **Live:** [tmarhguy.com](https://tmarhguy.com)
- **Stack:** React 18, TypeScript, Vite 7, Tailwind CSS, React Router
- **Deployment:** Static build, ready for GitHub Pages, Vercel, or any static host

---

## Features

- **Project showcase** — Hardware (ALU, MAC, RISC‑V) and software (UniBridge, MoMo, SVD) with detail pages
- **Certifications** — Coursera, Udemy, Onshape, academic credentials with verification links
- **About & story** — Background, skills, and recognition (BBC, AP, VOA, Deutsche Welle)
- **Document FAB** — Quick access to resume and cover letter
- **SEO & meta** — Canonical URLs, Open Graph, Twitter cards, structured data
- **Responsive** — Mobile-first layout with touch-friendly navigation

## Development

```bash
# Install
npm install

# Dev server (port 3001)
npm run dev

# Production build
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```

## Project Structure

```
portfolio/
├── public/           # Static assets (images, PDFs, certifications)
├── src/
│   ├── components/   # UI components, layout, features
│   ├── data/        # Projects, certifications data
│   ├── hooks/       # useMediaQuery, useInView, etc.
│   ├── pages/       # Route pages
│   └── utils/       # SEO helpers
├── index.html
└── vite.config.ts
```

---

## Deployment

The site builds to a static `dist/` folder. Deploy to:

- **GitHub Pages** — Push `dist` to `gh-pages` or use GitHub Actions
- **Vercel** — Connect repo, zero config
- **Netlify** — Same, or drag-and-drop `dist`
